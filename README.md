<h2>ğŸ“– Overview</h2>
<p>
<b>Attendance Calculator</b> is a powerful web application designed to help students track their academic presence with precision. Unlike simple calculators, this platform allows students to manage attendance for each subject individually. It provides a real-time analysis of their current standing, helping them stay ahead of their academic requirements.

<h2>ğŸš€ Key Features</h2>

<ul>
  <li><b>Subject-wise Tracking:</b> Add multiple subjects and input "Total Classes" and "Classes Attended" separately for each.</li>
  <li><b>Aggregated Percentage:</b> View a total attendance percentage calculated across all your subjects.</li>
  <li><b>Individual Status:</b> Monitor specific attendance percentages for every subject on a dedicated status page.</li>
  <li><b>Full CRUD Functionality:</b> Easily <b>Update</b> or <b>Delete</b> your attendance records whenever needed.</li>
  <li><b>Modern UI/UX:</b> Smooth navigation and high-quality visuals using <b>Framer Motion</b> and <b>Lottie React</b>.</li>
  <li><b>Cyber Aesthetic:</b> A sleek, dark-themed interface built with Tailwind CSS and DaisyUI.</li>
</ul>

<h2>ğŸ› ï¸ Installation & Setup</h2>

<ol>
  <li>Clone the repository:
    <pre>git clone https://github.com/prodip2005/attendance-calculator.git</pre>
  </li>
  <li>Install dependencies for both Client and Server:
    <pre># Install Client dependencies

    <ul>
